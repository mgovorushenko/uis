# Chat Scenario Board Prototype

Интерактивный прототип canvas-доски для сценариев обработки чатов.

## Запуск

```bash
python3 -m http.server 4173
```

После запуска откройте:

```text
http://127.0.0.1:4173/
```

## Что работает

- перетаскивание блоков по доске;
- zoom и pan через D3;
- создание новых операций из правой панели;
- выбор и редактирование блока;
- создание связей через коннекторы или панель свойств;
- удаление блоков;
- undo/redo для структурных действий;
- сохранение схемы в `localStorage`;
- восстановление схемы после перезагрузки.
